Tentang Project

Latar belakang, tujuan, dan metodologi pengembangan WebGIS Sorosutan

Latar Belakang

Kelurahan Sorosutan merupakan kawasan urban padat penduduk di Kecamatan Umbulharjo, Kota Yogyakarta. Pesatnya pertumbuhan bangunan mendorong kebutuhan akan sistem informasi spasial yang mampu menampilkan distribusi dan fungsi bangunan secara interaktif.

Tujuan WebGIS

  • Memetakan distribusi bangunan di Kelurahan Sorosutan
  • Mengidentifikasi fungsi bangunan secara spasial
  • Menyediakan informasi detail bangunan secara interaktif
  • Mendukung perencanaan tata ruang berbasis data

Teknologi

  • Mapbox GL JS v3.3 — visualisasi peta interaktif
  • GeoJSON — format data geospasial
  • HTML5, CSS3, JavaScript ES6

Kelas Pengguna

  • Admin — akses penuh, kelola website & data
  • Publik — akses baca, eksplorasi peta & informasi
